# Entangled Nodes

A mesmerizing multi-window experiment that synchronizes particle systems across different browser instances using the BroadcastChannel API and HTML5 Canvas.

## Features

- **Real-time Window Synchronization**: Particles react instantly to the relative positions of other browser windows on your screen.
- **Fluid Physics**: Nodes exhibit inertia based on window movement and gravitational attraction to other active "nodes."
- **Inter-window Visuals**: Dynamic "tendrils" and particle gravity effects bridge the gap between separate browser windows, creating a unified experience.
- **No Server Required**: Operates entirely on the client side using modern Web APIs.
